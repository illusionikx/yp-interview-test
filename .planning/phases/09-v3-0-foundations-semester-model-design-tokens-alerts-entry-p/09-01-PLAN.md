**Locked semantics this spec encodes (read before writing a single assertion):**

The `(year, number)` pair means exactly what `Section.year` / `Section.semester` mean:

09-RESEARCH.md's own `startsAt()`/`endsAt()` definitions, which all three agree on.

**CORRECTION to 09-RESEARCH.md — do not copy its ordinal() test.** Research proposes

proposed test only compares S2-2026 against S1-2027, so it passes while the inversion goes uncaught.
The correct relation is `ordinal() = year * 2 + (2 - number)`, but this spec must NOT simply assert
that arithmetic — it must assert the *property* that makes any formula error impossible to miss.
