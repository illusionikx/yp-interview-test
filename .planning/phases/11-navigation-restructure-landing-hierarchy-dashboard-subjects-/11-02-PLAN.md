  <behavior>
    - classesThisAndFuture: counts only Sections of the lecturer's assigned subjects whose
      composite semester ordinal is >= the current ordinal. A section in a past semester is excluded;
      a section in the current or a future semester is included. Crossing the Sep->Feb S1 rollover
      must NOT flip a past section into the count (this is why the naive comparison is banned).
    - enrolledVsSeats: totalSeats = SUM of capacity over assigned subjects' sections; enrolled =
      count of Enrolled enrollment rows on those sections. Zero seats yields a safe 0/0 (no divide-by-zero).
    - awaitingGrading: counts Attempts with status 'submitted' on exams belonging to assigned subjects.
    - subjects: only subjects assigned to the acting lecturer, each carrying sections_count and exams_count.
    - A lecturer with no assigned subjects sees all-zero cards and an empty subject list, not an error.
  </behavior>
  <action>
    Create app/Http/Controllers/Lecturer/HomeController.php with an index(Request) that returns
    view('lecturer.home', ...). Resolve the acting lecturer's assigned subject ids once via
    $request->user()->subjects()->pluck('subjects.id'). Compute exactly these bounded aggregates —
    every one a COUNT/SUM, never a PHP loop over relationships:
      - classesThisAndFuture: Section::whereIn('subject_id', $ids)
        ->whereRaw('(year * 2 + (2 - semester)) >= ?', [\App\Support\Semester::current()->ordinal()])->count().
        Add a code comment that the raw expression mirrors Semester::ordinal() and must not drift.
      - totalSeats: Section::whereIn('subject_id', $ids)->sum('capacity').
      - enrolledStudents: Enrollment::where('status', EnrollmentStatus::Enrolled)
        ->whereHas('section', fn ($q) => $q->whereIn('subject_id', $ids))->count().
      - awaitingGrading: Attempt::where('status', 'submitted')
        ->whereHas('exam', fn ($q) => $q->whereIn('subject_id', $ids))->count().
      - subjects: Subject::whereIn('id', $ids)->withCount(['sections', 'exams'])->orderBy('name')->get().
    Pass classesThisAndFuture, enrolledStudents, totalSeats, awaitingGrading, and subjects to the view.
    In routes/lecturer.php replace the home closure with
    Route::get('/', [\App\Http\Controllers\Lecturer\HomeController::class, 'index'])->name('home');
    (keep the name 'home' and the role:lecturer group unchanged).
  </action>
