<objective>
Build the student **class page** (TAK-07, TAK-08): a subject-scoped page reached from the Phase 11 student subject list's "Open class page" action. It shows the subject's detail plus the subject's exams in their own card — each exam marked not-yet-open / opened / closed, with a link to the result once taken or graded, and a Start button that is disabled once the student already has an attempt.

Purpose: This closes the student's navigation into the take-exam flow and fixes v2.0's sharpest defect (a result page that existed, was tested, but was unreachable) by making the taken/graded marker a real link.

Output: A new `Student\ClassPageController@show(Subject)`, its route `student.subjects.class`, the `student/subjects/class.blade.php` view, the two retargeted home links, and Feature tests. This is presentation over already-shipped enrollment/visibility/attempt backend — no enforcement logic is added or weakened.
</objective>
