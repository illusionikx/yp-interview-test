Notable defects caught **before** completion and fixed in-milestone:
- **P7 CR-01** — cross-subject exam-assignment leak: an exam could be assigned to, and become
  takeable via, a section of an unrelated subject. Broke the core value ("the right exam reaches
  the right student"). Fixed at the root (`AssignExamRequest` subject-scoped rule + subject-filtered UI).
- **P8 CR-01** — ENR-04 raced across *sibling* sections of the same subject/term; the lock now
  spans the whole invariant.
- **P8 CR-02** — a student orphaned mid-attempt by withdrawal/**rejection** could not navigate back
  to their live attempt (exam dropped out of `visibleTo()` and the page 403'd), contradicting
  AVL-04/AVL-05. Fixed with an ownership-driven "Resume exam" path.
- **Phase-5-era gap** — the student result page had **no inbound UI link** at all; the feature
  shipped verified-by-existence but was unreachable. Fixed in-phase (`78eb271`).
