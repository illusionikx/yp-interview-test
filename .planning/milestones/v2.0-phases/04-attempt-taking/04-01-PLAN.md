## Phase Goal (MVP user story)

**As a** student, **I want to** take a time-limited exam whose answers save as I go and submit exactly once, **so that** my work is captured reliably and cannot be gamed, duplicated, or lost.

This phase is delivered as sequential vertical slices: 04-01 (this plan) is the RED contract; 04-02 makes "start an exam and see the questions" real (single-attempt + no-leakage + server timer); 04-03 makes "answers save as I go" real (autosave); 04-04 makes "submit / time's up" real (submit + live countdown + auto-submit).
</objective>
