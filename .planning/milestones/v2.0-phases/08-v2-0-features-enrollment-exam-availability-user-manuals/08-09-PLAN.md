       section per subject each semester; a section that is full or outside its enrollment window
       has no Apply button, and applying anyway is refused with a message explaining why.
    2. "Withdrawing" — the Withdraw link, the confirmation dialog, the fact that you can withdraw

       lecturer has graded every open-text answer.

    3. "Authoring and Assigning an Exam with an Availability Window" — create a draft exam, add

       **Call out the draft-only constraint explicitly and in plain language: set the availability

    4. "Grading" — finding an exam's results, grading open-text answers within the question's
